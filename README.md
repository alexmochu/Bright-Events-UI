## Install

1. Clone the repo: 
```
$ git clone [URL]
```
2. Install dependencies.
```
$ npm install
```
3. Run the app:
```
$ npm start
```
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
## Running tests

```
$ npm test
```
